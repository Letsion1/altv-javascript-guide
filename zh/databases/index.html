# 介绍

有很多数据库，它们都有不同的用途.

根据您的需求选择, MongoDB, MySQL, PostgreSQL 甚至 MariaDB 将适用于您的服务器.

如果您是编程新手, 我强烈建议 MongoDB 因易于使用.
