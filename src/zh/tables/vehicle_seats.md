# 载具座位

此部分代表玩家可以坐在的座位.

| 索引 | 描述                             |
| ---- | -------------------------------- |
| -1   | 驾驶位                           |
| 0    | 副驾驶                           |
| 1    | 后排左乘客                       |
| 2    | 后排右乘客                       |
| 3    | 左后乘客 (大于4个座位的任何座位) |
| 4    | 右后乘客 (大于4个座位的任何座位  |
